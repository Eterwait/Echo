Mechanical structure

Note: You can find all hardware files and pictures in the Echo repository /Hardware folder.

Echo has a complex mechanical structure, consisting of hundreds of components. The setup includes two mechanical arms (though it is possible to use just a single arm), which are attached to a table (we use a welding table) and suspended using counterbalances on a structure made of aluminum profiles. The ECHO arm consists of several typical joints and a joystick. The components of the joints and joystick are produced using 3D printers from PLA and TPU92A plastics and assembled with fasteners, couplings, and shafts. The 3D model of ECHO has hierarchical numbering system for avoiding confusion comfortable usage and complience with lists of components which are avalable bellow.
